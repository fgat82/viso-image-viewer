## 核心功能一览

**格式支持够用**

支持 PNG、JPG、HEIF、HEIC、DNG、RAW（含 ARW 等）、BMP 等日常常见格式，摄影师拍回来的原片直接拖进来就能看，不用先转格式。

**图片信息随叫随到**

按需显示拍摄参数——镜头型号、色彩配置文件、分辨率尺寸、拍摄地点地图。不需要的时候一键收起，界面干净如初。这个功能对整理旧照片特别有用，不用专门打开 Lightroom 才能看 EXIF 信息。

**文件夹连续浏览**

在一个文件夹里，左右方向键直接翻图，不用频繁回到 Finder 去点下一张。逻辑简单，但用起来顺手得很。

**内置格式转换**

可以直接把图片转换成 PNG、JPEG 或 BMP 格式，省去单独开转换工具的麻烦。对设计师来说，出图、给甲方、格式转换这一套在 Viso 里就能顺手搞定。

**Touch Bar 和 Apple Silicon 原生支持**

如果你有带 Touch Bar 的 MacBook Pro，Viso 支持在 Touch Bar 上显示快捷操作。Intel 和 M 系列芯片都是原生适配，不走 Rosetta 转译，打开速度和响应都很快。
